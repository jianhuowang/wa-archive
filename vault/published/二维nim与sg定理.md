# 二维 Nim 与 SG 定理

## 从一维到二维

坐标 (x, y)，每步选一个方向（横或纵）走 1 或 2 格。

两个方向互不影响 → **两个独立子游戏**。

## SG 定理（Sprague-Grundy 定理）

> **独立子游戏的总 SG = 各子游戏 SG 的异或（XOR）**

```
一维 SG(x) = x % 3
一维 SG(y) = y % 3
SG(总) = (x % 3) ⊕ (y % 3)
```

- XOR = 0 → 必败
- XOR ≠ 0 → 必胜

## 这个具体问题的答案

mod 3 只有 0、1、2。XOR = 0 的唯一情况是两个数相等：

```
(x % 3) == (y % 3)  →  先手必败
等价于 (x - y) % 3 == 0
```

## 关键步骤

> **先取余（算 SG），再 XOR。** 不要跳过算 SG 直接 XOR 原数。
