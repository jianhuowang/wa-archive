# Nim 游戏的 XOR 原理

## SG 定理

**独立子游戏的总 SG = 各子游戏 SG 的异或（XOR）。**

## XOR = 0 的本质

XOR = 二进制逐列数 1 的个数，不进位：

**非零例** `0 ⊕ 1 ⊕ 2`：
```
0 = 00
1 = 01
2 = 10
⊕ = 11 = 3
第 0 位：0+1+0 = 1（奇数 → 1）
第 1 位：0+0+1 = 1（奇数 → 1）
→ XOR ≠ 0 → 有得修 → 必胜
```

**归零例** `1 ⊕ 2 ⊕ 3`：
```
1 = 01
2 = 10
3 = 11
⊕ = 00 = 0
第 0 位：1+0+1 = 2（偶数 → 0）
第 1 位：0+1+1 = 2（偶数 → 0）
→ XOR = 0 → 平衡态 → 必败
```

- **每列 1 的个数全为偶数** → XOR = 0 → **平衡态**
- **存在某列 1 的个数为奇数** → XOR ≠ 0 → 可以修

## 如何修复不平衡

当前总 XOR = k ≠ 0。

1. 找 k 的**最高位 1** 所在的列
2. 必有一堆在那列也是 1，设该堆 SG = g
3. 目标：让该堆 SG 变成 g' = g ⊕ k
4. 因为最高位 1 被翻成 0，g' < g

**关键**：mex 的定义保证了——若某状态的 SG = g，则它一定存在后继状态，其 SG 为 0, 1, 2, …, g-1 中的**任意值**。既然 g' < g，就存在合法走法让该子游戏的 SG 精确变成 g'。

修改后新 XOR = k ⊕ g ⊕ g' = k ⊕ g ⊕ (g ⊕ k) = 0。平衡态甩给对手。

对手下一步必然打破平衡 → 你再修 → 循环到你拿最后一步。

## 正确步骤

> **先算各子游戏的 SG（取余等），再 XOR。** 不要跳过算 SG 直接 XOR 原数。
