## 为什么想做这个博客

题解、课堂笔记和项目复盘散落在不同地方时，很难形成长期积累。WA Archive 想做的，是给这些内容一个稳定、可检索、愿意反复打开的家。

## 技术选型

第一版采用 **Astro + Markdown/MDX + GitHub Pages**。内容直接保存在仓库里，没有数据库，也没有需要维护的后端服务。

## Codex 帮我做了什么

Codex 根据明确的边界创建了页面、内容集合、深色模式、代码高亮和自动部署流程。更重要的是，项目从第一天开始就有真实内容，而不是一副空壳。

## 我踩了哪些坑

最危险的坑不是代码，而是试图在第一版加入所有功能。评论、后台和复杂动画都被留到了真正需要它们的时候。
