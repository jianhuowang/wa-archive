# 一维 Nim 的 SG 推导

## 问题

站在位置 $n$（还剩 $n$ 步能走）。每步走 1 或 2 格。走到 0 动不了 → 输。

哪些位置是必败的？

## 方法：从 0 开始逐个推

核心规则：
> **输** ⇔ 你能走到的所有位置**全是赢**
> **赢** ⇔ 能走到的位置里**至少有一个输**

```
n=0: 动不了                            → 输
n=1: 只能走到 0（输）                   → 赢
n=2: 能走到 1（赢）、0（输）            → 赢（走 2 步到 0）
n=3: 能走到 2（赢）、1（赢）            → 输（怎么走对面都赢）
n=4: 能走到 3（输）、2（赢）            → 赢（走 1 步到 3）
n=5: 能走到 4（赢）、3（输）            → 赢（走 2 步到 3）
n=6: 能走到 5（赢）、4（赢）            → 输
```

**规律：$n \bmod 3 = 0$ 时必败。**

## 用 SG 值重算

用 [[SG值的定义与mex运算]] 的公式：

| n | 能走到 | 那些位置的 SG | mex | SG |
|---|--------|--------------|-----|----|
| 0 | — | — | mex{} | 0 |
| 1 | 0 | {0} | mex{0} | 1 |
| 2 | 1, 0 | {1, 0} | mex{1,0} | 2 |
| 3 | 2, 1 | {2, 1} | mex{2,1} | 0 |
| 4 | 3, 2 | {0, 2} | mex{0,2} | 1 |
| 5 | 4, 3 | {1, 0} | mex{1,0} | 2 |
| 6 | 5, 4 | {2, 1} | mex{2,1} | 0 |

SG 序列：0, 1, 2, 0, 1, 2, 0, … → **循环节 = 3**。

**结论：$SG(n) = n \bmod 3$。** SG = 0 的位置 = $n \bmod 3 = 0$，✓ 吻合。
