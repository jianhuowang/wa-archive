# SG 值的定义与 mex 运算

## 一句话

**SG 值** 是一个量化局面（位置）是赢是输的非负整数。

## 定义

$$
SG(n) = \operatorname{mex}\{\,SG(m) \mid \text{从 }n\text{ 能一步走到 }m\,\}
$$

> 一个位置的 SG = 从它**能走到的所有位置**的 SG 值集合，取 mex。

## mex 是什么？

**mex** = **m**inimum **ex**cludant（最小排除值）

从 0 往上数，**第一个不在**这个集合里的非负整数。

```
mex{}      = 0    （空集，0 不在→答 0）
mex{0}     = 1    （0 在→往上，1 不在→答 1）
mex{0,1}   = 2    （0 在, 1 在, 2 不在→答 2）
mex{1,2}   = 0    （0 不在！→答 0）
mex{0,2}   = 1    （0 在, 1 不在→答 1）
```

「第一个不在集合里的」是 mex 的全部含义。没有别的。

## SG = 0 意味着什么？

| SG 值 | 含义 |
|-------|------|
| **SG = 0** | **必败**（这个位置谁走谁输） |
| **SG ≠ 0** | **必胜**（这个位置先手能赢） |

为什么？
- SG = 0 意味着你**走不到**任何一个 SG = 0 的位置（mex 不会漏掉 0）
- 对手只能走到 SG ≠ 0 的位置
- 对手总有办法把你送回 SG = 0，直到你无法动弹

## 实用性质：SG 值的上界

> **SG(n) ≤ 从 n 能走到的状态总数（出度）**

原因：mex 作用于一个有 d 个元素的集合，最大值就是 d（集合恰为 {0,1,…,d−1} 时）。

每步只有 3 种走法 → 任何位置的 SG ≤ 3。编写代码分配数组大小时有用。
